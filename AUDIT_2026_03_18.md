# Conductor Audit — March 18, 2026

## Current State
- **Version:** v3.2.1 | 69 Swift files | ~21,100 lines
- **Status:** All core features implemented. No stubs, no TODOs, no crashes.
- **Architecture:** Per-window state isolation, NDJSON streaming, async/await (no Combine)

## What's Already World-Class
- Context preservation pipeline (auto-extract decisions/files/tasks, pre-compaction snapshots, reinjection)
- Multi-agent orchestration (4 patterns: supervisor, pipeline, consensus, swarm)
- Permission management with learning rules
- Token/cost tracking with graduated warnings
- Keyboard-first design (every action has a shortcut)
- HSL-based luminance slider (single control, infinite themes)
- Session continuity (cross-session handoff via artifacts)
- MCP catalog (8 servers, one-click install)
- Dev tools integration (CodeRabbit, SwiftLint, Periphery, Fastlane)
- Gemini panel (Cmd+Shift+G)

## Opportunities for Autonomy & Power

### High Impact
1. Pre-compaction hook never wired (CompactionEngine has code, never called)
2. Model routing is suggestion-only (could auto-apply for simple messages)
3. No auto-retry on failures (lastPrompt saved but never retried)
4. No "autonomous mode" for extended unattended operation
5. No file watchers / auto-build on code changes
6. Session closeout doesn't summarize (avoids cost, but optional summary useful)

### Medium Impact
7. Permission learning requires 3 approvals (could be 2 for low-risk)
8. Agent teams hidden behind env var (could be a toggle)
9. No batch operations / workflow templates
10. No auto-suggest next actions after task completion

### UX Polish
11. Budget display showed "$0 cap" (FIXED this session)
12. No floating chat export (Cmd+E defined but no view)
13. No app icon (still Xcode default)
14. No notarization for v3.2.x
